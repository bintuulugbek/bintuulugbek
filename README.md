## 🌟 Featured Project

### [`claude-agent-workflow`](https://github.com/bintuulugbek/claude-agent-workflow) — a portable operating system for AI coding agents

A spec-driven rig that decides **where a human must sign off** and **where an AI
can safely run unattended**: two trust profiles (HITL / earns-AFK), three human
gates (SHAPE · TAG · JUDGE), a "Test Trust" discipline that treats every green
check as a claim to verify, three specialized agents, and an autonomous **Ralph
loop** that works a Kanban board on its own.
